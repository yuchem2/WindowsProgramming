
# HitBlow

+ Date: 2022.11.26-2022.12.08 

## 프로그램 설명

해당 프로그램은 학부 2학년 수업인 윈도우즈프로그래밍 및 실습의 기말 프로젝트로써 제작된 프로그램이다.  
닌텐도 스위치에 존재하는 51 Worldwide Games에 수록되어 있는 Hit and Blow를 모작해 제작하였다.


## 기획단계

게임의 전체 흐름은 총 3가지로 나뉘게 된다. 전체 프로그램을 3개로 나누어 제작을 진행하였다. 
1. 게임 초기
2. 게임 진행
3. 게임 종료

### 게임 초기 
<img src="https://user-images.githubusercontent.com/101711808/235669199-6ccb3bc7-7908-46fe-a97c-a7a450b4408b.png" width="700" height="350">

게임 초기 화면은 총 2개의 버튼이 존재하는 화면을 가진다. 두개의 버튼은 각각 P1, P2 버튼이다.

### 게임 진행
<img src="https://user-images.githubusercontent.com/101711808/235670091-eaf5ecbe-b104-4ae0-b3bb-add686156c69.png" width="700" height="350">

게임 진행 화면은 가장 중요한 기능을 포함하는 화면으로써 많은 기능이 필요하다.  
맨 아래 위치한 색깔 칸은 공을 선택하는 창이다. 1인용의 경우 좌우 방향키로만 이동할 수 있으며 2인용의 경우 1P는 a, d를, 2P는 좌우 방향키를 이용해 이동할 수 있다.  

화면 중앙에서 부터 위치하고 있는 숫자를 포함한 칸은 선택한 공을 놓는 창이다. 위의 조그만 4개의 원은 Hit, Blow 여부를 나타낸다.  
위치한 칸은 1인용의 경우 상하 방향키로만 이동할 수 있으며, 2인용의 경우 1P는 w, s를, 2P는 상하 방향키를 이용해 이동할 수 있다.  
입력이 완료되면 Hit, Blow 여부가 작은 네모에 표시가 된다. Hit은 주황이고, Blow는 하양이다.

맨 우측에 위치한 칸은 답을 숨기고 있는 창으로, 결과를 맞추거나 8번의 기회(2인용의 경우 각 4번)를 전부 소모하고 못 맞출 경우 공개된다.

### 게임 종료
<img src="https://user-images.githubusercontent.com/101711808/235671585-4efad0de-344f-4261-a580-e97a518f5238.png" width="700" height="350">

위에 기술한 대로 결과를 맞추거나 8번의 기회를 전부 소모한다면 답이 공개되고, 그에 해당하는 결과를 추가적인 윈도우를 이용해 출력한다. 그 후 이용자에게 종료 혹은 재시작을 입력받을 수 있다. 


### 구현 방법

각 이미지는 따로 비트맵을 활용하지 않고, 직접 HDC를 이용해 그리기를 이용해 구현하고자 한다.  
게임 중 입력은 w, a, s, d와 방향키 상화좌우만 받는다. 게임 종료나 게임 초기화면에서는 마우스 입력을 통해 각 선택을 받는다.  
각 구현은 앞서 Tetris 프로젝트의 방법을 참고하기로 하였다. 

추가적으로 메뉴와 엑셀러레이터를 활용해 편하게 사용할 수 있도록 구현할 예정이다. 


## 구현완료

### 게임 초기
<div>
  <img src="https://user-images.githubusercontent.com/101711808/235673688-01224a4a-e745-45de-ac57-79bd27afa659.png" width="450" height="350">
  <img src="https://user-images.githubusercontent.com/101711808/235674071-37bd9d67-99be-466f-8f14-226e36532538.png" width="450" height="350">
</div>

총 4개의 버튼이 존재한다. 1P, 2P, Exit, HELP 버튼이 존재한다.  
1P, 2P 버튼은 1인, 2인 게임을 결정한다. Exit 버튼은 게임 종료키로, ESC 버튼을 통해서도 가능하고, 메뉴의 게임-종료를 통해서도 가능하다.  
HELP 버튼은 게임의 설명을 담은 윈도우를 말한다. 게임의 룰을 설명한다. 이 버튼 또한 Ctrl+H을 통해서도 가능하며 메뉴의 도움말-도움말보기를 통해서도 가능하다.  
우측 사진은 도움말 보기를 눌렀을 때 화면이다. 

### 게임 진행
<img src="https://user-images.githubusercontent.com/101711808/235674299-02fd3dee-bfbd-4f15-bd99-e7dd71956c46.png" width="450" height="350">

해당 위 사진은 게임 진행 중 일시정지버튼(P)를 눌렀을 때 발생하는 화면인다.  
게임 조작은 w, a, s, d, 방향키를 이용해 이동할 수 있다. Enter를 입력하면 Hit, Blow 판단을 진행한다.  
이를 최대 8번 반복하며 색깔을 맞추게 된다. 


### 게임 종료
<img src="https://user-images.githubusercontent.com/101711808/235674466-d01f1cc6-a996-404c-b2ea-4b8c28440ce1.png" width="450" height="350">

해당 화면은 게임 종료 화면이다. 주어진 색깔 조합을 맞추거나 주어진 기회를 모두 소진하는 경우 발생하게 된다.  
게임이 끝나는 경우 End 윈도우가 발생하며 게임 종료에 대한 정보를 화면에 출력하게 된다. 재시작 또는 종료를 선택할 수 있다. 


## 실행방법

HitBlow/Debug/HitBlow.exe 를 실행하면 된다.
